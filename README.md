​Bộ hướng dẫn này sẽ giúp bạn cài đặt hoàn chỉnh môi trường chơi Minecraft Java Edition với đầy đủ các mod tối ưu hóa và công cụ hỗ trợ điều khiển từ xa.
​📥 Bước 1: Tải Các Tệp Cần Thiết
​Bạn cần tải về đủ 4 tệp sau đây để bắt đầu:
​PojavLauncher.apk: Trình khởi chạy Minecraft Java.
​MT_Manager.apk: Công cụ quản lý tệp và giải nén (khuyên dùng).
​RustDesk.apk: Ứng dụng chia sẻ màn hình để nhận hỗ trợ hoặc điều khiển từ xa.
​MinecraftMods.zip: Gói dữ liệu gồm .minecraft và controlmap.
​🛠️ Bước 2: Cài Đặt Ứng Dụng
​Sau khi tải về, tiến hành cài đặt lần lượt các file có đuôi .apk:
​Cài đặt PojavLauncher.
​Cài đặt MT Manager.
​Cài đặt RustDesk.
​🎮 Bước 3: Khởi Tạo PojavLauncher
​Trước khi bỏ mod, bạn cần tạo "bộ khung" cho game:
​Mở ứng dụng PojavLauncher.
​Nhập tên người chơi: Nhập nickname bạn muốn sử dụng.
​Chọn phiên bản: Tìm và chọn đúng dòng Fabric Loader 1.21.4.
​Nhấn Chơi để ứng dụng tải dữ liệu gốc (Sau khi vào đến menu game thì có thể thoát ra để làm bước tiếp theo).
​📂 Bước 4: Giải Nén & Ghi Đè Dữ Liệu (Quan Trọng)
​Đây là bước sử dụng MT Manager để cài đặt Mod và sơ đồ phím:
​Mở MT Manager.
​Cửa sổ bên trái: Tìm đến file MinecraftMods.zip. Nhấn vào và chọn Giải nén.
​Cửa sổ bên phải: Truy cập theo đường dẫn sau:
​Android/data/net.kdt.pojavlaunch/files/
​Thao tác: Kéo toàn bộ các thư mục bên trong file zip đã giải nén (thường là .minecraft và controlmap) từ bên trái Ghi đè sang bên phải.
​Nếu máy hỏi, chọn "Ghi đè tất cả" (Replace all).
​🖥️ Bước 5: Kết Nối & Trải Nghiệm
​Chia sẻ màn hình: Mở app RustDesk, thiết lập ID và Password để chia sẻ màn hình nếu bạn cần hỗ trợ hoặc muốn điều khiển từ PC.
​Vào Game: Quay lại PojavLauncher và bấm Chơi.
​Thưởng thức: Game đã được tích hợp sẵn Mod tối ưu và sơ đồ phím điều khiển.
​💡 Lưu ý nhỏ
​Hiển thị file ẩn: Nếu trong MT Manager không thấy thư mục .minecraft, hãy vào cài đặt của app và bật "Show hidden files".
​Cấu hình RAM: Nên chỉnh mức RAM trong PojavLauncher từ 2GB - 4GB để chơi mượt mà nhất.
