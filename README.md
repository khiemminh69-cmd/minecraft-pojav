# mods
1. ​Cloth Config API (cloth-config-17.0.144-fabric.jar): Thư viện hỗ trợ giao diện cấu hình cho các mod khác.
2. ​Fabric API (fabric-api-0.119.4+1.21.4.jar): Thư viện cốt lõi bắt buộc phải có để chạy hầu hết các mod trên Fabric.
3. ​FerriteCore (ferritecore-7.1.3-fabric.jar): Mod tối ưu hóa, giúp giảm lượng RAM mà game tiêu thụ.
4. ​FPS Display (fpsdisplay-4.3.0+1.21.4.jar): Hiển thị chỉ số FPS (khung hình trên giây) trực tiếp trên màn hình.
​5. Krypton (krypton-0.2.8.jar): Tối ưu hóa ngăn xếp mạng (stack network) để giảm giật lag khi chơi multiplayer.
6. ​Lithium (lithium-fabric-0.15.3+mc1.21.4.jar): Mod tối ưu hóa hiệu suất tổng thể (game logic, AI, vật lý) mà không làm thay đổi tính năng gốc của game.
7. ​Meteor Client (meteor-client-1.21.4-42.jar): Một bộ công cụ (thường được gọi là "hack client") hỗ trợ nhiều tính năng tiện ích và PvP.
​8. ModernFix (modernfix-fabric-5.20.3+mc1.21.4.jar): Cải thiện tốc độ tải game và sửa các lỗi hiệu năng nhỏ.
9. ​Mod Menu (modmenu-13.0.4.jar): Thêm một nút "Mods" vào menu chính để bạn xem danh sách và tùy chỉnh các mod đã cài.
10. ​Sodium (sodium-fabric-0.6.10+mc1.21.4.jar): Mod tăng FPS cực kỳ hiệu quả bằng cách thay đổi engine render của game.
