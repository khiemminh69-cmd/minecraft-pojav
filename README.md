# mods
Dưới đây là bảng liệt kê chi tiết:
| Tên File | Tên Mod | Công dụng chính |
|---|---|---|
| fabric-api-0.119.4+1.21.4.jar | **Fabric API** | Thư viện lõi bắt buộc để hầu hết các mod Fabric khác có thể hoạt động được. |
| ferritecore-7.1.3-fabric.jar | **FerriteCore** | Giảm đáng kể lượng RAM mà Minecraft sử dụng bằng cách tối ưu hóa các mô hình dữ liệu trong game. |
| krypton-0.2.8.jar | **Krypton** | Tối ưu hóa ngăn xếp mạng (network stack) để giảm độ trễ (lag) và cải thiện tốc độ truyền dữ liệu khi chơi server. |
| lithium-fabric-0.15.3+mc1.21.4.jar | **Lithium** | Một bản mod tối ưu hóa "toàn diện" giúp cải thiện tốc độ xử lý của game (tick rate) mà không làm thay đổi cơ chế gốc. |
| meteor-client-1.21.4-42.jar | **Meteor Client** | Một bộ công cụ (client) cung cấp nhiều tính năng hỗ trợ người chơi, từ tự động hóa đến các tiện ích mở rộng trong game. |
### **Lưu ý nhỏ:**
 * **Sự tương thích:** Tất cả các file đều khớp với phiên bản **1.21.4**, đây là một điểm cộng vì sẽ tránh được lỗi xung đột phiên bản.
 * **Mục đích:** Bộ mod này rất mạnh về mảng tối ưu hóa (FerriteCore, Krypton, Lithium), giúp máy chạy mượt hơn và ít tốn tài nguyên hơn khi treo máy hoặc tham gia các server đông đúc.
